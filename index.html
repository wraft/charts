Welcome to the official Helm chart repository for Wraft - a comprehensive application deployment solution with improved organization and maintainability.

📦 Available Charts

  • wraft - Main application deployment chart (v0.1.5)

🛠️ Usage

Add the Helm Repository

helm repo add wraft https://wraft.github.io/charts helm repo update

Install the Chart

helm install my-wraft wraft/wraft

Install with Custom Values

helm install my-wraft wraft/wraft -f my-values.yaml
